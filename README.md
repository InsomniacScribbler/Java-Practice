# Java-Practice

**My practice repository for Java programming and OOPs (Object-Oriented Programming) basics.**

This repository contains code snippets, examples, and practice problems that cover core Java concepts and foundational OOP principles. It serves as a hands-on learning and revision tool as I build my Java proficiency.

- ✅ Java Syntax & Basics  
- ✅ Object-Oriented Programming Concepts  
  - Classes & Objects  
  - Inheritance  
  - Polymorphism  
  - Abstraction  
  - Encapsulation  
- ✅ Method Overloading & Overriding  
- ✅ Constructors  
- ✅ Access Modifiers  
- ✅ Static vs Non-static  
- ✅ Interfaces vs Abstract Classes  
- ✅ Exception Handling  
- ✅ Java Memory Model Basics
